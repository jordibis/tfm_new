# main.py

Aquí es oden modificar els hiperparàmetres i canviar el tipus de mètode i dataset per correr el que es desitgi i obtenir els resultats

Codi per fer el gràfic de l'exactitud de tots el mètodes
# feature_graph.py
Codi utilitza per fer les figures de t-sne i de gradcam. Es necessita d'un model ja entrenat.
# AsyCon.py
Codi per preentrenar el model amb el mètode d'aprenentatge contrastiu asimètric
# requirements.txt
Archiu amb totes les llibreries que es necessiten instalar per fer funcionar correctament el codi.
